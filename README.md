# Palindrome Checker Application

## Objective
The objective of this project is to create a Java application that checks whether a given string is a palindrome.

## Project Procedure
1. Created a Java class named PalindromeCheckerApp.
2. Used Scanner class to take input from the user.
3. Reversed the string using logic.
4. Compared the original string with the reversed string.
5. Displayed the result.

## Project Flow
User Input → Reverse String → Compare → Display Result

## Use Case
- Helps understand string manipulation
- Demonstrates conditional logic in Java
- Useful for beginners learning programming
